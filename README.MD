# Python Pixelsuft GUI
My first python gui lib.<br />
Based on [pysdl2](https://pypi.org/project/PySDL2/) <br />
[Page On GitHub](https://github.com/Pixelsuft/pypixgui) <br />
![Screenshot](https://github.com/Pixelsuft/pypixgui/raw/main/screenshot.png)
# Installing
Windows:
```
python -m pip install pypixgui
```
For Linuxoids:
```
sudo apt-get install python3-tk python3-dev python3-sdl2 -y
sudo pip3 install pypixgui
```
